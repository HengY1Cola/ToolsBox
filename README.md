## 前言

欢迎来到**数据结构与算法**世界 Welcome to the world of data structures and algorithm.

数据结构与算法可以说是学习计算机的**基础中的基础**。

在我的看法中有几处好处：

1. 过面试的**敲门砖**🧱，你可以不是精通但是你得会
2. 考试的**拉分**项目，学分高，好拿分
3. 使用某种语言来实现数据结构与算法，可以快速掌握
4. 巧用算法，在实战中可能会有好的思路
5. 锻炼脑子，即使工作拧螺丝，但是仍然能脑瓜子聪明
